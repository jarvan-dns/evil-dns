# DNS隐蔽信道数据集
由于现存DNS隐蔽信道（即DNS Tunnel）流量多为隧道工具流量（例如iodine），为应对DNS隐蔽信道类威胁，特开源工具流量+各平台查找到的真实攻击流量+自生成流量（挑选后使用），用于AI模型研究（仅供科研使用）。现更新部分，后续文章完结后将持续更新。数据集为实验室内各位小伙伴共同贡献，后续沟通后署名。
